Quick Reference

Top-level keys:

  • api_keys — groq, newsdata, telegram
  • web — host, port, debug, secret_key
  • use_preset_nifty500 — true/false
  • symbols — list of NSE tickers (.NS)
  • fundamentals — cache_hours, filters
  • screen — lookback, top_n, weights, thresholds
  • news — enabled, days_back, max_articles
  • groq — model, max_tokens, temperature
  • schedule — list of job objects
  • telegram — enabled, bot_token, chat_id
  • logging — level, max_bytes, backup_count
Symbol Format

NSE symbols need .NS suffix:

symbols:
  - RELIANCE.NS
  - TCS.NS
  - INFY.NS
  - HDFCBANK.NS

Set use_preset_nifty500: true to use the built-in ~250 stock universe instead.

Schedule Format 
schedule:
  - name: morning_scan
    cron: "45 7 * * 1-5"
    tasks:
      - fundamentals
      - screen
      - news
      - ai_analysis
      - telegram_report
  - name: evening_check
    cron: "45 15 * * 1-5"
    tasks:
      - screen
      - news
      - ai_analysis

Cron format: minute hour dom month dow
